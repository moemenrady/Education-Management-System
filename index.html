نظام إدارة منظومة تعليمية

نظام ويب متكامل لإدارة السناتر والأكاديميات التعليمية، يهدف إلى تنظيم وإدارة جميع العمليات اليومية بفاعلية واحترافية، مع توفير لوحات تحكم وتقارير دقيقة لاتخاذ القرارات.

الأسبوع الأخير مخصص لتجربة النظام وتدريب المستخدمين

نطاق العمل (Scope of Work)

1. إدارة المواد والكتب

  • إضافة، تعديل، وبحث عن المواد الدراسية.
  • تسجيل وصف المادة ورابط WhatsApp.
  • إدارة الكتب الدراسية وربطها بالمواد.
  • تسجيل سعر التكلفة وسعر بيع الكتب.

2. إدارة المجموعات

  • إنشاء وتعديل المجموعات وربطها بالمواد.
  • تحديد يوم وموعد الحصة.
  • تحديد حالة المجموعة (نشطة / متوقفة).

3. ملف الطالب الشامل

  • إدارة بيانات الطلاب (الاسم، الكود، هواتف التواصل، المدرسة).
  • ملف مخصص لكل طالب يضم (سجل التجديدات، الحضور، الفواتير).
  • مشاركة ملف الطالب عبر رابط أو PDF لولي الأمر.

4. إدارة الاشتراكات

  • تسجيل اشتراك جديد وتجديد الاشتراكات.
  • متابعة عدد مرات التجديد وتاريخ الاشتراك.
  • تحديد قيمة الاشتراك وربطه بالمجموعة.

5. الحضور الذكي بالباركود

  • إنشاء باركود فريد لكل طالب (إمكانية الطباعة والمشاركة).
  • تسجيل الحضور آلياً وسريعاً عبر قارئ الباركود (Scanner).
  • سجل حضور وغياب مفصل لكل حصة.
  • تقليل الأخطاء البشرية وتسريع الإجراءات.

6. الحسابات والفواتير

  • إصدار فواتير تلقائية للاشتراكات والتجديدات.
  • طباعة الفواتير ومعاينتها.
  • إدارة بنود المصروفات (إيجار، مرتبات، صيانة، إلخ).
  • تسجيل تفاصيل وحركة المصروفات.

7. الامتحانات والموظفين

  • إدارة امتحانات المواد (تسجيل وحفظ).
  • إدارة بيانات موظفي السنتر (الوظيفة، الراتب، التواصل).
  • ربط الموظف بحساب للدخول (اختياري).

الصلاحيات وإدارة النظام

مدير النظام (Admin)

صلاحية كاملة لإدارة النظام، المستخدمين، كافة البيانات، والاطلاع المفتوح على جميع التقارير المالية والإدارية.

السكرتير (Secretary)

إدارة الطلاب، الاشتراكات، تسجيل الحضور، إصدار الفواتير، تسجيل المصروفات، والاطلاع على التقارير التشغيلية فقط.

التقارير المتاحة

  • تقرير الطلاب والمواد
  • تقرير المجموعات والاشتراكات
  • تقارير الاشتراكات (الجديدة/المنتهية)
  • تقرير الحضور والغياب
  • تقرير الإيرادات والمصروفات
  • تقرير الأرباح والفواتير

مزايا إضافية للتشغيل

  • لوحة تحكم إحصائية (Dashboard) للمؤشرات.
  • بحث سريع وذكي داخل جميع شاشات النظام.
  • قابلية مشاركة وتصدير البيانات.
  • واجهة مستخدم بسيطة ولا تتطلب خبرة تقنية.
